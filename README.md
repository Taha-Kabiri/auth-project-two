#   Installation and setup:

   1.  Install MongoDB on your system
   2.  Installing Node.js
   3.  Gate installation
   4.  **Cloning the repository:**
   5.  ```bash
   git clone [https://github.com/Taha-Kabiri/auth-project-two.git](https://github.com/Taha-Kabiri/auth-project-two.git)
    cd auth-project-two
       ```
   6.   -Installing used packages: " npm install " (Type this in the terminal vs code)
   7.  Project implementation : " npm start "  (Type this in the terminal vs code)



   -----------------------------------------------------------------------------------------------------------------------
   View the output by typing the following URL in your browser:
   1 :  http://127.0.0.1:3000/api/user     -> all users
   2 :  http://127.0.0.1:3000/api/auth/register   -> Register page
   3 :  http://127.0.0.1:3000/api/auth/login  -> login page
   4 :  http://127.0.0.1:3000/api/dashbord   -> dashbord user ( If you have a token, you can access the dashboard from this route for up to 24 hours after each login )
